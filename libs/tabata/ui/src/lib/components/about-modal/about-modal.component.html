Traditional Tabata (what you’re building)

A classic Tabata workout is built from high-intensity work intervals (20s) and short recoveries (10s), repeated for 8 rounds. In this app, each “main” block follows that structure.

Warmup, main workout, cooldown

  • Warmup — Low-intensity cardio + safe stretching. It prepares joints and helps your muscles wake up (typically 5–10 minutes).
  • Main workout — Tabata blocks (work/rest intervals). Your level + primary goal shape the exercise categories, while target muscles match the selection.
  • Cooldown — Lower intensity only. It uses stretching exercises and avoids high-impact categories (typically 3–7 minutes).

How to create a workout

  • Open the Workouts tab and start creating a new workout.
  • Fill “Workout info”: name, description, level, primary goal, available equipment, main target, and optional secondary targets.
  • Generate with AI (in create mode): the AI uses your level + primary goal + equipment + targets to select exercises and build warmup/main/cooldown.
  • Adjust manually if needed: use the Warmup/Main/Cooldown phase tabs to review and refine exercises.

Best results with the AI generator

  • Match the vibe: write a description that reflects your goal (for example: quick cardio burn, strength focus, or explosion/plyometric emphasis).
  • Be consistent: choose a level that matches how hard you want the workout to feel (downward compatibility is enforced: harder tier never gets “up-leveled” moves).
  • Choose correct equipment: if you select only “body only”, the AI will avoid exercises requiring equipment.
  • Cover targets: set main + secondary targets so the selector can choose exercises that overlap those muscles.

The generator also prevents duplicate exercise selection across warmup, main, and cooldown, so the workout stays varied.

Workout player (during execution)

When you start playing, the player guides you through each phase: Warmup first, then Main blocks (each exercise shows its interval work/rest), and finally Cooldown.

When the workout ends you’ll see a completion screen (and you can restart or go back).

Workout info options (quick reference)

  • Name + description — used as the “brief” so the AI understands what to build.
  • Level — difficulty tier; controls which exercise difficulty is allowed (downward compatibility).
  • Primary goal — maps to exercise categories (Strength → strength, Cardio → cardio, Explosion → plyometrics).
  • Available equipment — filters exercises to what you have.
  • Main target + Secondary targets — tells the generator which muscles should overlap with chosen exercises.
