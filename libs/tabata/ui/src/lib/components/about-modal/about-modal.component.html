Demo app

When viewed from web, switch to mobile view (browser DevTools or responsive mode) for the best experience.

This is a demonstration application for learning and example code only. It is not intended for commercial use. Use it to explore Tabata workout flows, AI-generated workouts, and modern Angular patterns.

Tested only on: web and Android (OnePlus 10 Pro). Other devices and iOS have not been tested.

Technologies

  • Angular — Signals, standalone components, control flow (@if, @for)
  • Ionic — UI components, tabs, modals, action sheets
  • Firebase — Authentication and optional backend
  • Nx — Monorepo, libs, build pipeline
  • Google Gemini — AI workout generation via API
  • REST API — Generate-workout endpoint (serverless function)
  • Environment — Secret API keys via env vars (e.g. GEMINI_API_KEY)
  • Vercel — Deployment and serverless functions
  • Upstash — Serverless data / Redis when used
  • Capacitor — Native mobile builds (iOS / Android)

State management

NgRx-style facades and stores (e.g. AuthFacade, workout state). Angular Signals for reactive UI and computed values. Data persisted via Firebase.

How to use

  • Dashboard — Last and popular workouts, quick play.
  • Workouts — Browse, create, or edit. Use “Generate with AI” on the create flow for an AI-generated structure.
  • History — Workout history, most used, and favorites.
  • Profile — Display name, email, password. Log out from here.
